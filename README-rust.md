# Rust Integration

Ensure that rust-analyzer is installed in the normal location `~/.cargo/bin/rust-analyzer`.

Optional (I don't use this anymore. I'm using plain `neovim/nvim-lspconfig`)
  See https://github.com/mrcjkb/rustaceanvim

# Key bindings

Add rust specific key bindings to `after/ftplugin/rust.lua`

# Check nvim state

Check the nvim state in
```
ls -latr ~/.local/state/nvim/
```
